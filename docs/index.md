# Sweet Compose Documentation

This directory contains the main documentation for Sweet Compose. It is organized by audience and topic.

- For a high-level product and architecture overview, start with:
  - `overview.md`
  - `architecture/runtime-and-layout.md`
- For contributors, see:
  - `development/contributing.md`
  - `development/testing.md`
- For usage guides:
  - `guides/drawing.md` — drawing API reference
  - `README.md` — minimal hello-world example
- For what works and what doesn't:
  - `limitations.md` — **single source of truth** for current and long-term limitations; read this before assuming an API works end-to-end
- For roadmap, status, and planning:
  - `roadmap/long-term-direction.md` — long-term evolution plan (SWT‑first vs Material‑like modes, drawing backends)
  - `roadmap/drawing-foundation.md` — drawing/graphics foundation plan; **source of truth for drawing work**
  - `roadmap/test-determinism.md` — test determinism analysis
- For background reading (not Sweet-specific, but useful context):
  - `background/jetpack-compose-layout.md`
  - `background/swt-layout.md`
